實作使用GitHub Pages & Hexo 建立個人Blog

發表於 2020-07-31 更新於 2020-08-02 分類於 使用 GitHub Pages & Hexo 建立個人Blog

前置作業都準備好之後就開始我們的實作吧~

資料夾的名字會是你的網址，所以可以取個有意義的

建立出來的資料夾內容

安裝git佈署套件

Hexo設定檔

打開專案下的_config.yml，修改裡面的內容

(1) Blog的基本資訊(line 6 ~12)

(2) 第16行修改成你自己的Github Pages的網址

root的地方記得要加上 /<your_project_name>/ 否則主題會顯示不出來

(3) 修改佈署至Git上的設定(line 98~)

type改為git、並且加上repo以及branch
repo的路徑為：https://github.com/<yourname>/<your_project_name>.git

(4) 接著在cmd視窗下指令

(5) 輸入網址就可以看到使用Hexo佈署上去的樣子啦~

網址為我們在設定檔的第16行所設定的
url: http://<yourname>.github.io/<your_project_name>

補充

大部分的教學在建立project時通常會將project_name取作.github.io，因此只要輸入http://.github.io就可以看到自己的github page了，差異會出現在3-2步驟，其餘皆相同。
更多詳細的教學可以到參考的文章去看看~
